# slam-website
A local-first Android safety application for generating SLAM and Incident Reports with photo support. Built with Kotlin and Jetpack Compose.

## Getting Started

1. Clone this repository to your **local hard drive** (not a USB drive)
2. Open the project in Android Studio or your preferred IDE
3. Build and run the application

## Troubleshooting

Having issues with Git or the development environment? Check our [Troubleshooting Guide](TROUBLESHOOTING.md) for solutions to common problems, including:

- **"git add ." not working** - Especially when working from USB drives
- Build and dependency issues
- File path and permission problems
